# Korean_Lip_Reading

### 주제 선정 배경
의사소통에 어려움을 겪는 유형의 장애인 중 청각장애인이 5만명 정도로 가장 높은 수치를 보인다. 또한 소음이 있는 환경이나 거리가 먼 경우 음성 대화가 어렵기 때문에 어떠한 상황에서도 원활한 의사소통이 가능하게 하는 보조 도구로써 시스템을 설계하였다.
